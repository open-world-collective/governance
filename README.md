# Governance

This repository holds governance, charter, and transparency documents for Open World Collective.

## What this is

An open governance and accountability workspace for a transparency-first, community-governed virtual world project.

## Current status

Early-stage governance and operations scaffolding.

## What exists today

- Governance draft
- Charter draft
- Decision log
- Transparency report template
- Governance roadmap issues

## Scope note

Open World Collective is currently an engineering collaboration effort and not a legal foundation entity.
